# Script de extração de Reviews

## Requisitos

node instalado

## Utilização

1 - Instale os pacotes

```bash
npm install
```

2 - Execute o script

```bash
npm start
```

3 - Aguarde a execução do script

4 - Verifique o resultado da extração nos arquivos `googlePlay_reviews_govBr.csv` e `googlePlay_reviews_govBr.json`
